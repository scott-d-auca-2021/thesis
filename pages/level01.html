Нахождение дискриминанта и корней в квадратных уравнениях


Квадратное уравнение — это уравнение вида \(ax^2 + bx + c = 0\), где \(a\), \(b\) и \(c\) — известные числа, причем \(a \neq 0\).


Простыми словами:

Число, которое стоит возле \(x^2\) будем называть \(a\)

Число, которое стоит возле \(x\) будем называть \(b\)

Число, которое стоит отдельно без \(x\) будем называть \(с\)

Если \(x^2\) стоит без числа, значит \(a = 1\). Если \(x\) стоит без числа, значит \(b = 1\)


Дискриминант квадратного уравнения помогает определить количество и характер корней уравнения. Он вычисляется по формуле:

\[D = b^2 - 4ac\]


Рассмотрим разные случаи:


1) Когда \(a\) отрицательный: \(-2x^2 + 3x + 1 = 0\)


В данном случае: \(a = -2; b = 3; c = 1\), следовательно \[D = 3^2 - 4 * (-2) * 1 = 9 + 8 = 17\]


2) Когда \(b\) отрицательный: \(-2x^2 - 3x + 1 = 0\)


В данном случае: \(a = -2; b = -3; c = 1\), следовательно \[D = (-3)^2 - 4 * (-2) * 1 = 9 + 8 = 17\]


Из-за того, что \(b\) берется в квадратную степень, он обратно становится положительным и ответ не меняется.


3) Когда \(c\) отрицательный: \(x^2 - x - 1 = 0\)


В данном случае: \(a = 1; b = -1; c = -1\), следовательно \[D = (-1)^2 - 4 * 1 * (-1) = 1 + 4 = 5\]


4) Когда \(c\) = 0: \(x^2 - 3x = 0\)


В данном случае: \(a = 1; b = -3; c = 0\), следовательно \[D = (-3)^2 - 4 * 1 * 0 = 9 - 0 = 9\]


В зависимости от значения дискриминанта:

  • Если \(D > 0\), уравнение имеет два различных решения.
  • Если \(D = 0\), уравнение имеет одно решение.
  • Если \(D < 0\), уравнение не имеет решений.

Нахождение корней/решений квадратного уравнения:

Корни (решения) квадратного уравнения (если \(D \geq 0\)) можно найти с помощью даннной формулы:

\[x_{1,2} = \frac{{-b \pm \sqrt{D}}}{{2a}},\]

где знак \(\pm\) означает, что каждый корень нужно вычислить отдельно:


  • \(x_1 = \frac{{-b + \sqrt{D}}}{{2a}}\)
  • \(x_2 = \frac{{-b - \sqrt{D}}}{{2a}}\)

Пример 1:


\[4x^2 - 4x + 1 = 0\]

\[a = 4; b = -4; c = 1\] \[D = (-4)^2 - 4 * 4 * 1 = 16 - 16 = 0\]

\(D = 0\), значит у нас будет 1 решение:


\[x_{1,2} = \frac{{-(-4) \pm \sqrt{0}}}{{2*4}};\]

\[x_{1,2} = \frac{{4 \pm 0}}{{8}};\]

\[x_{1,2} = \frac{{4}}{{8}} = \frac{{1}}{{2}};\]


Пример 2:


\[2x^2 - 7x + 3 = 0\]

\[a = 2; b = -7; c = 3\] \[D = (-7)^2 - 4 * 2 * 3 = 49 - 24 = 25\]

\(D > 0\), значит у нас будет 2 решения


\[x_1 = \frac{{-(-7) + \sqrt{25}}}{{2*2}};\]

\[x_1 = \frac{{7 + 5}}{{4}} = \frac{12}{4} = 3;\]

\[x_2 = \frac{{-(-7) - \sqrt{25}}}{{2*2}};\]

\[x_2 = \frac{{7 - 5}}{{4}} = \frac{2}{4} = \frac{1}{2};\]
