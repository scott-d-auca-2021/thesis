Квадрат теңдемелерде дискриминантты жана чечимдерди табуу


Квадраттык теңдеме - бул \(ax^2 + bx + c = 0\) түрүндөгү теңдеме, мунда \(a\), \(b\) жана \(c\) - белгилүү сандар, ал эми \(a \neq 0\).


Жөнөкөй сөздөр менен:

\(x^2\) жанындагы санды \(a\) деп атайбыз

\(x\) жанындагы санды \(b\) деп атайбыз

\(x\)сиз турган санды \(c\) деп атайбыз

Эгерде \(x^2\) сансыз турса, анда \(a = 1\). Эгерде \(x\) сансыз турса, анда \(b = 1\)


Дискриминант квадраттык теңдеменин тамырларынын санын жана мүнөзүн аныктоого жардам берет. Ал төмөнкү формула менен эсептелет:

\[D = b^2 - 4ac\]


Ар кандай учурларды карап көрөлү:


1) Эгерде \(a\) терс болсо: \(-2x^2 + 3x + 1 = 0\)


Ушул учурда: \(a = -2; b = 3; c = 1\), демек \[D = 3^2 - 4 * (-2) * 1 = 9 + 8 = 17\]


2) Эгерде \(b\) терс болсо: \(-2x^2 - 3x + 1 = 0\)


Ушул учурда: \(a = -2; b = -3; c = 1\), демек \[D = (-3)^2 - 4 * (-2) * 1 = 9 + 8 = 17\]


\(b\) чарчы дөңгөлөкчө алынгандыктан, ал кайра оңго айланат жана жооп өзгөрбөйт.


3) Эгерде \(c\) терс болсо: \(x^2 - x - 1 = 0\)


Ушул учурда: \(a = 1; b = -1; c = -1\), демек \[D = (-1)^2 - 4 * 1 * (-1) = 1 + 4 = 5\]


4) Эгерде \(c\) = 0 болсо: \(x^2 - 3x = 0\)


Ушул учурда: \(a = 1; b = -3; c = 0\), демек \[D = (-3)^2 - 4 * 1 * 0 = 9 - 0 = 9\]


Дискриминанттын маанисине жараша:

  • Эгерде \(D > 0\) болсо, теңдеме эки ар түрдүү чечимди алат.
  • Эгерде \(D = 0\) болсо, теңдеменин бир чечими болот.
  • Эгерде \(D < 0\) болсо, теңдеме чечимдери жок.

Квадраттык теңдеменин тамырларын/чечимдерин табуу:

Тамырлар (чечимдер) квадраттык теңдемеден (эгерде \(D \geq 0\) болсо) төмөнкү формула менен табылат:

\[x_{1,2} = \frac{{-b \pm \sqrt{D}}}{{2a}},\]

\(\pm\) белгиси ар бир тамырды айрым алып чыгуу керектигин билдирет:


  • \(x_1 = \frac{{-b + \sqrt{D}}}{{2a}}\)
  • \(x_2 = \frac{{-b - \sqrt{D}}}{{2a}}\)

Мисал 1:


\[4x^2 - 4x + 1 = 0\]

\[a = 4; b = -4; c = 1\] \[D = (-4)^2 - 4 * 4 * 1 = 16 - 16 = 0\]

\(D = 0\), демек бизде 1 чечим бар:


\[x_{1,2} = \frac{{-(-4) \pm \sqrt{0}}}{{2*4}};\]

\[x_{1,2} = \frac{{4 \pm 0}}{{8}};\]

\[x_{1,2} = \frac{{4}}{{8}} = \frac{{1}}{{2}};\]


Мисал 2:


\[2x^2 - 7x + 3 = 0\]

\[a = 2; b = -7; c = 3\] \[D = (-7)^2 - 4 * 2 * 3 = 49 - 24 = 25\]

\(D > 0\), демек бизде 2 чечим бар:


\[x_1 = \frac{{-(-7) + \sqrt{25}}}{{2*2}};\]

\[x_1 = \frac{{7 + 5}}{{4}} = \frac{12}{4} = 3;\]

\[x_2 = \frac{{-(-7) - \sqrt{25}}}{{2*2}};\]

\[x_2 = \frac{{7 - 5}}{{4}} = \frac{2}{4} = \frac{1}{2};\]
